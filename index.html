极简高效，让待办重回专注

专为现代浏览器打造的微软待办增强插件。采用 Local-First 架构，提供丝滑的同步体验与深度定制的功能扩展。

Chrome 商店 Edge 商店 GitHub Release

极致性能体验

采用 Local-First 架构，UI 操作实现零延迟反馈，彻底告别原生网页版的加载等待感。

优雅 UI 交互

适配 Windows 11 设计语言。支持自动暗黑模式、智能任务图标匹配与丝滑的侧边栏交互。

双区调优加速

内置中国区与国际区加速配置，针对性解决国内环境下登录缓慢与同步断连的顽疾。

跨标签协同锁

基于 Cross-tab Locking 机制，无论开启多少浏览器窗口，均能确保同步任务有序进行。

智能差分同步

仅同步变更数据，配合指数重试算法，在网络波动环境下亦能保证任务百分百触达云端。

离线沙箱引擎

支持全功能离线编辑。无网络时自由增删改查，待联网后自动执行智能差分合并同步。

可视化任务导图

一键将待办列表转化为思维导图，支持多级子任务展示，让复杂项目脉络清晰可见。

Secondary 隐私锁

支持为特定列表设置本地二级密码。即使电脑处于解锁状态，你的私密任务依然固若金汤。

隐私与安全，我们的核心原则

我们深知待办事项往往包含个人敏感信息。MS-ToDo 采用以下技术保障：

数据自主控制：敏感数据存储在本地 IndexedDB。
透明同步流向：仅与微软官方 API 和你的专用代理通信。
本地锁保护：私密列表支持 secondary 密码访问。
